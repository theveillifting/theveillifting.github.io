# theveillifting website

## Add or change the Canva banner

1. Export your Canva banner as a PNG.
2. Rename it to `banner.png`.
3. Put it in `assets/images/`, replacing the existing `banner.png`.

## Add a blog post

1. Copy `NEW-BLOG-POST-TEMPLATE.md`.
2. Paste the copy into the `_posts` folder.
3. Rename it like this: `2026-05-06-my-post-title.md`.
4. Change the `title` and `date` at the top.
5. Write the post under the second `---`.

GitHub Pages will publish it automatically after you commit the change.
